Hello! My name is Cameron, and I see Linux differently

But first, what is Linux?

Linux is, for the guys who make it, a kernel. it's the base piece of code, that mediates between the silicon and the software you see. Yeah, it's pretty important. Linux is just one kind of kernel, Microsoft Windows, and Apple OS X are both Operating Systems. So, Linux is just a different way to do things.

OK, what's good about it?

Linux was made to share! Linux has a special licensing --as does most of the software that Linux uses-- that allows you special freedoms not afforded by other types of Intelectual Property.

  1. Linux lets you use it however you want.
    • No "Digital Handcuffs" or "Digital Restrictions Management"
  2. You can change Linux to work however you want.
    • the source code is made freely available.
  3. you can share linux
    • sharing is important, right?
  4. and you can share your changes
    • because this is open source

How do I make your Linux experience better?

Linux can be daunting. Wireless drivers, terminal scripts, scary error messages, and unexplained bugs all can discourage use. I'm here to help. I can set it up for you, fix it if something breaks, and help you use it better. I'm your very own "Geek Squad". Even better, I help with Linux for free!
